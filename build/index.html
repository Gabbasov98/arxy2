Квартира для Полины Гагариной
В самой работе над проектом самой сложной задачей оказалось совмещение нового и старого интерьеров. Ремонт делался только для коридора и гостиной, а остальная часть квартиры осталась нетронутой. Было много интересных задач в концепции самого дизайна. Например, совместить в гостиной лёгкий парижский стиль и брутальные акценты.

Беседка для Димы Биланы
Зона барбекю хорошо сочетается с расположенной рядом альпийской горкой, искусственным водоемом, цветочными клумбами и другими элементами приусадебного интерьера. Помимо эстетической составляющей необходимо обратить внимание ещё на 3 основных фактора: пожарная безопасность, затенение место отдыха, направление ветра.

Ландшафтный проект для Нюши

Интерьер для Филиппа Киркорова

Кроме зала, в здании находится ещё помещения, где по просьбе заказчика мы разместили SPA и комнату для отдыха.
